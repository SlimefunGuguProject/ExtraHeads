# ExtraHeads 额外头颅

ExtraHeads 是一个 Slimefun4 附属插件。
你需要安装 Slimefun4 才能让此插件运行。

该附属向游戏添加了更多的生物头颅。击杀生物有几率掉落其头颅，可以使用斩首之剑来增加掉落头颅的几率。
